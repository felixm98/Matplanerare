# Matplanerare
Hjälper dig planera ditt kalori och mineral/vitamin intag samt sätter upp en måltidsplan

## Kom igång

### Skapa din egen version

För att skapa din egen version av detta projekt, följ dessa steg:

1. Ladda ner projektfilerna (inte klona med git)
2. Öppna en terminal och navigera till projektmappen
3. Initiera ett git repository och pusha till ditt eget GitHub-konto:

```bash
cd /sökväg/till/matplanerare
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/DITT-ANVÄNDARNAMN/matplanerare.git
git push -u origin main
```

**Obs:** Byt ut `DITT-ANVÄNDARNAMN` med ditt faktiska GitHub-användarnamn och `/sökväg/till/matplanerare` med sökvägen till din projektmapp.
